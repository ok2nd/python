# 比率指定トリミング

画像を指定した縦横比でトリミングする Windows 用アプリです。

---

## 起動方法

**「起動.bat」をダブルクリック**してください。
初回起動時に必要なライブラリが自動でインストールされます。

---

## 必要環境

- Windows 10 / 11
- Python 3.10 以上（https://www.python.org/）
- インターネット接続（初回ライブラリ取得時）

---

## 使い方

1. **画像をドラッグ＆ドロップ**してアプリウィンドウに放す
   （または「📂 ファイルを開く」ボタン）
2. 左パネルの**比率一覧から比率を選択**
   → トリミング枠が自動で中央に表示されます
3. **枠をドラッグして位置・サイズを調整**
   - 枠の中をドラッグ → 移動
   - 四隅のハンドルをドラッグ → 比率を維持してリサイズ
   - 枠の外をドラッグ → 新しい枠を作成
4. **「💾 保存」ボタン**で保存
   → 元のファイルと同じフォルダに `元の名前trim.拡張子` で保存されます

---

## 比率設定のカスタマイズ

### アプリ内で編集
「⚙ 比率設定を編集」ボタンを押して編集します。

### ファイルで編集
アプリと同じフォルダの `trim_ratios.txt` をメモ帳などで編集します。

**形式:**
```
名前:横比率:縦比率
```

**例:**
```
# # で始まる行はコメント
正方形:1:1
横16:9:16:9
A4縦:210:297
```

---

## 機能一覧

| 機能 | 詳細 |
|------|------|
| ドラッグ＆ドロップ | 画像ファイルを直接ドロップ |
| 比率維持トリミング | 選択比率で枠が自動制約 |
| ICC カラープロファイル保持 | 色管理情報を引き継ぎ |
| 選択比率の記憶 | アプリ再起動後も維持 |
| 比率設定の編集 | アプリ内 / テキストファイル両方 |
| 保存形式 | 元のファイルと同じ形式・同じフォルダ |

---

## 対応画像形式

JPEG / PNG / TIFF / BMP / WebP
